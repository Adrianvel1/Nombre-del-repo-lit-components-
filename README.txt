Proyecto: Sistema de Login y Pokémon con LitElement

Descripción:
Este proyecto permite a los usuarios iniciar sesión con credenciales predefinidas 
y visualizar información de un Pokémon asociado a su usuario.

Instrucciones de uso:
1. Iniciar sesión con uno de los usuarios disponibles.
2. Una vez dentro, el sistema mostrará detalles del Pokémon asignado al usuario.

Los usuarios y contraseñas que asigne son: 
- charmeleon: "1234",
   
- bulbasaur: "5678",

- squirtle: "91011",

- pikachu: "1213";
